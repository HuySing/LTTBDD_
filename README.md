# Dự Án Lập Trình Di Động (Mobile Programming)

## Giới Thiệu

Lập trình di động (Mobile Programming) là quá trình phát triển ứng dụng dành cho các thiết bị di động như smartphone và máy tính bảng. Các ứng dụng này có thể được phát triển cho các nền tảng khác nhau như Android, iOS và nền tảng đa dạng như React Native hoặc Flutter.

Mục tiêu của dự án này là cung cấp một cái nhìn tổng quan về lập trình di động, từ đó giúp người học nắm bắt các kỹ thuật và công cụ cần thiết để phát triển ứng dụng di động.

## Mục Tiêu Dự Án

Dự án này sẽ giúp bạn:
- Hiểu về các nền tảng di động phổ biến: Android, iOS, React Native, Flutter.
- Cài đặt môi trường phát triển ứng dụng di động.
- Tạo ra các ứng dụng di động cơ bản cho Android và iOS.
- Làm quen với các công cụ phát triển như Android Studio, Xcode, và các framework di động đa nền tảng.

## Công Nghệ Sử Dụng

- **Android Development**: Sử dụng Java hoặc Kotlin để phát triển ứng dụng cho Android.
- **iOS Development**: Sử dụng Swift để phát triển ứng dụng cho iOS.
- **React Native**: Framework JavaScript cho phép phát triển ứng dụng di động đa nền tảng.
- **Flutter**: Framework của Google để phát triển ứng dụng di động đa nền tảng.

## Các Tính Năng Của Dự Án

- **Giao diện người dùng (UI)**: Hướng dẫn xây dựng các giao diện đơn giản, dễ sử dụng.
- **Xử lý sự kiện**: Thực hiện các hành động dựa trên sự kiện người dùng như click nút, kéo thả.
- **Quản lý dữ liệu**: Kết nối ứng dụng với cơ sở dữ liệu để lưu trữ và truy xuất thông tin.
- **Tính năng đa nền tảng**: Xây dựng ứng dụng có thể chạy trên cả Android và iOS.

## Hướng Dẫn Cài Đặt

1. **Cài đặt Android Studio:**
   - Truy cập [Android Studio](https://developer.android.com/studio) và tải về phiên bản mới nhất.
   - Cài đặt theo hướng dẫn trên trang web.

2. **Cài đặt Xcode (chỉ dành cho macOS):**
   - Truy cập [Xcode](https://developer.apple.com/xcode/) để tải và cài đặt.
   
3. **Cài đặt Flutter (cho phát triển đa nền tảng):**
   - Truy cập [Flutter](https://flutter.dev/docs/get-started/install) và làm theo các bước cài đặt.

4. **Cài đặt React Native (nếu muốn phát triển đa nền tảng):**
   - Truy cập [React Native](https://reactnative.dev/docs/environment-setup) và làm theo hướng dẫn.

## Cách Sử Dụng

Sau khi cài đặt các công cụ cần thiết, bạn có thể bắt đầu phát triển ứng dụng di động theo các bước sau:

1. **Tạo dự án mới**:
   - Android Studio: `File` -> `New` -> `New Project`
   - Xcode: `File` -> `New` -> `Project`
   - React Native: `npx react-native init ProjectName`
   - Flutter: `flutter create project_name`

2. **Chạy ứng dụng**:
   - Android Studio: Chọn thiết bị giả lập hoặc thiết bị thật và nhấn nút `Run`.
   - Xcode: Chọn thiết bị và nhấn nút `Run`.
   - React Native: Sử dụng lệnh `npx react-native run-android` hoặc `npx react-native run-ios`.
   - Flutter: Sử dụng lệnh `flutter run`.
